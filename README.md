# bencoding.cr

Bencoding serialization library using annotations.

## Installation

1. Add the dependency to your `shard.yml`:
```yaml
dependencies:
  bencoding:
    github: light-side-software/bencoding.cr
```
2. Run `shards install`

## Usage

```crystal
require "bencoding"
```
